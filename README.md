# 📊 Unemployment Analysis with Python

## 📌 Project Overview

This project analyzes unemployment rate data using Python to understand employment trends, identify seasonal patterns, and evaluate the impact of the COVID-19 pandemic. The analysis includes data cleaning, exploratory data analysis (EDA), and visualization to generate meaningful insights that can support economic and social policy decisions.



## 🎯 Objectives

- Analyze unemployment rate data.
- Clean and preprocess the dataset.
- Explore unemployment trends using visualizations.
- Investigate the impact of COVID-19 on unemployment.
- Identify seasonal and regional unemployment patterns.
- Present insights that can support economic and social policies.



## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook



## 📂 Dataset

The dataset contains unemployment statistics, including:

- Region
- Date
- Frequency
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)
- Area (Urban/Rural)

## 🔍 Key Insights

- Unemployment increased significantly during the COVID-19 pandemic.
- Urban areas experienced higher unemployment fluctuations than rural areas.
- Some regions consistently recorded higher unemployment rates.
- Monthly analysis revealed seasonal variations in unemployment.
- Employment gradually recovered after lockdown restrictions were eased.
